# Fire Emblem Tool
Very much WIP

## Running
``` bash
npm run watch
```
Note! Automatic watch still not doe

## Usage
Go to `localhost:8081` in your browser
You can use characterName as a query parameter (e.g. `http://localhost:8081/?characterName=Crom`) to search for different characters.
